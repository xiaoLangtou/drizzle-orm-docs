# Drizzle <> PostgreSQL</>
<Prerequisites>
- 数据库[连接基础](/docs/connect-overview)与 Drizzle
- gel-js [基础知识](https://github.com/geldata/gel-js)
</Prerequisites>

Drizzle 通过 `gel-js` 客户端原生支持 Gel 连接。

#### 步骤 1 - 安装包
<Npm>
drizzle-orm gel
-D drizzle-kit
</Npm>

#### 步骤 2 - 初始化驱动并执行查询
<CodeTabs items={["gel", "gel with config"]}>
```typescript copy
// 确保安装了 'gel' 包
import { drizzle } from 'drizzle-orm/gel';

const db = drizzle(process.env.DATABASE_URL);
 
const result = await db.execute('select 1');
```
```typescript copy
// 确保安装了 'gel' 包
import { drizzle } from "drizzle-orm/gel";

// 你可以指定任何 gel 连接选项
const db = drizzle({
  connection: {
    dsn: process.env.DATABASE_URL,
    tlsSecurity: "default",
  },
});

const result = await db.execute("select 1");
```
</CodeTabs>

如果你需要提供你现有的驱动：

```typescript copy
// 确保安装了 'gel' 包
import { drizzle } from "drizzle-orm/gel";
import { createClient } from "gel";

const gelClient = createClient();
const db = drizzle({ client: gelClient });

const result = await db.execute('select 1');
```
