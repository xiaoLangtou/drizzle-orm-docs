# 开始使用 Drizzle 和 PostgreSQL

<Prerequisites>
  - **dotenv** - 用于管理环境变量的包 - [了解更多](https://www.npmjs.com/package/dotenv)
  - **tsx** - 用于运行 TypeScript 文件的包 - [了解更多](https://tsx.is/)
  - **node-postgres** - 用于查询 PostgreSQL 数据库的包 - [了解更多](https://node-postgres.com/)
</Prerequisites>

Drizzle 原生支持使用 `node-postgres` 和 `postgres.js` 驱动程序连接 PostgreSQL。

在本入门示例中，我们将使用 `node-postgres`。如果你想了解更多连接 PostgreSQL 的方式，请查看我们的 [PostgreSQL 连接](/docs/get-started-postgresql) 页面

如果你还没有 PostgreSQL 数据库并想创建一个用于测试，你可以使用我们的 Docker 中设置 PostgreSQL 的指南。

PostgreSQL Docker 指南可在[这里](/docs/guides/postgresql-local-setup)获取。按照指南设置好数据库，生成数据库 URL（在指南中有说明），然后继续接下来的步骤
