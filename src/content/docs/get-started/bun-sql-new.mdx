# 开始使用 Drizzle 和 Bun:SQLite

<Prerequisites>  
  - **bun** - JavaScript 全能工具包 - [点击了解更多](https://bun.sh/)
  - **Bun SQL** - 用于操作 PostgreSQL 数据库的原生绑定 - [点击了解更多](https://bun.sh/docs/api/sql)
</Prerequisites>

<Callout type='error'>
在版本 `1.2.0` 中，Bun 在执行并发语句时存在问题，如果您尝试同时运行多个查询，可能会导致错误。
我们已经创建了一个 [GitHub issue](https://github.com/oven-sh/bun/issues/16774) 供您跟踪。一旦修复，您就不会再在 Bun 的 SQL 端遇到此类错误
</Callout>
