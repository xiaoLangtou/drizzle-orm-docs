# 开始使用 Drizzle 和 SingleStore

<Prerequisites>  
  - **dotenv** - 用于管理环境变量的包 - [了解更多](https://www.npmjs.com/package/dotenv)
  - **tsx** - 用于运行 TypeScript 文件的包 - [了解更多](https://tsx.is/)
  - **mysql2** - 用于查询 MySQL 数据库的包 - [了解更多](https://github.com/sidorares/node-mysql2)
</Prerequisites>

要在 SingleStore 数据库中使用 Drizzle，你应该使用 `singlestore` 驱动

根据**[官方网站](https://github.com/sidorares/node-mysql2)**的说明，
`mysql2` 是一个专注于性能的 Node.js MySQL 客户端。

Drizzle ORM 通过 `drizzle-orm/singlestore` 包原生支持 SingleStore 数据库的 `mysql2`。
