# 使用 Drizzle Kit 进行迁移
<Prerequisites>
- Drizzle 和 `drizzle-kit` 入门 - [阅读此处](/docs/get-started)
- Drizzle schema 基础 - [阅读此处](/docs/sql-schema-declaration)
- 数据库连接基础 - [阅读此处](/docs/connect-overview)
- Drizzle 迁移基础 - [阅读此处](/docs/migrations)
- Drizzle Kit [概述](/docs/kit-overview) 和 [配置文件](/docs/drizzle-config-file)
- `drizzle-kit generate` 命令 - [阅读此处](/docs/drizzle-kit-generate)
- `drizzle-kit migrate` 命令 - [阅读此处](/docs/drizzle-kit-migrate)
</Prerequisites>

Drizzle 允许你生成空的迁移文件，以便编写自定义的 SQL 迁移，用于当前 Drizzle Kit 不支持的 DDL 修改或数据填充，然后你可以使用 [`drizzle-kit migrate`](/docs/drizzle-kit-migrate) 命令运行这些迁移。

```shell
drizzle-kit generate --custom --name=seed-users
```

```sql
INSERT INTO "users" ("name") VALUES('Dandrew');
```
